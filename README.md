# MossionX

Landing page premium desenvolvida para posicionamento digital de marcas de alto padrão.

## ✨ Sobre o projeto

A MossionX foi criada para transmitir:
- autoridade
- sofisticação
- posicionamento premium
- impacto visual cinematográfico

O projeto utiliza uma estética minimalista escura com elementos dourados, animações suaves e foco total em percepção de valor.

- CSS3

- Motion Effects
- Scroll Reveal Animations
